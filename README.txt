Introduction

This code is used to create a movie website displaying movie's title, poster image and YouTube trailer.

Filling Movie's Data

If you want to create a new movie tile, kindly follow these steps:

    Make sure you have python installed on your machine.
    Open entertainment_center.py using IDLE.
    Create a new instanse of the class Movie located in the file media.py.
    Pass the movie's title first, followed by poster image URL then the YouTube trailer link.
    Add the name of the instance to the movies list.
    Save and run the code.
